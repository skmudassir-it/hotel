# 🏨 Hotel Booking Backend

> Express + MongoDB API for hotel room management.

**🎉 New modern frontend available:**

### 👉 [hotel-frontend-next.vercel.app](https://hotel-frontend-next.vercel.app)

The new version features:
- 🏛️ Elegant luxury hotel landing page
- 🛏️ 6 room cards with images, prices, and amenities
- 🏊 Amenities showcase (pool, spa, dining, fitness)
- 📞 Contact & booking CTA section
- ✨ Dark gold/cream theme with Framer Motion animations
- ⚡ Built with Next.js 16 + Tailwind CSS

**Source:** [github.com/skmudassir-it/hotel-frontend-next](https://github.com/skmudassir-it/hotel-frontend-next)
